# Релиз 1.6.3

Дата: 2026-05-17

## Основное

- Выпущен корректирующий Android-релиз после выявленной регрессии верхнего правого меню.
- Версия продукта увеличена до `1.6.3`, Android `versionCode` увеличен до `163`.

## Исправления

- Верхнее правое меню больше не использует UI Kitten `OverflowMenu`, который зависит от popover/measure слоя и нестабилен после обновления React Native.
- Меню действий перенесено на обычный `Modal` с safe-area позиционированием справа сверху.
- Сохранены действия меню: «Обновить», переключение светлой/тёмной темы и «Выйти».
- Добавлена статическая регрессионная проверка, чтобы меню не вернулось на `OverflowMenu`.
