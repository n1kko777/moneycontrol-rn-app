# Релиз 1.6.1

Дата: 2026-05-15

## Основное

- Обновлена мобильная версия продукта до `1.6.1`.
- Android `versionCode` увеличен до `161` для корректной публикации новой APK-сборки.
- Приложение обновлено до Expo SDK 55 и актуальной связки React Native, совместимой с текущим Xcode.
- Конфигурация EAS Update приведена в консистентное состояние: добавлен `expo-updates`, а источник версии EAS зафиксирован как локальный.
- Устранены npm audit уязвимости: проверка проходит с результатом `0 vulnerabilities`.

## Исправления

- Исправлен web crash в селекторах операций, связанный с UI Kitten Autocomplete и `findNodeHandle` в React Native Web.
- Стабилизированы сценарии выбора счета, категории, профиля и тегов в E2E-тестах.
- Удалено устаревшее поле Expo config, из-за которого `expo-doctor` не проходил полностью.

## Проверки

- `npm ci`
- `CI=1 npx expo install --check`
- `npx expo-doctor`
- `npm audit`
- `npm run lint`
- `npm run build:web`
- `CI=1 EXPO_PUBLIC_API_URL=http://127.0.0.1:8000 EXPO_WEB_PORT=8082 npx playwright test tests/e2e/auth-smoke.spec.ts`
